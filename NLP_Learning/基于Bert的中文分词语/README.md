本实践使用icwb2(http://sighan.cs.uchicago.edu/bakeoff2005/)数据集进行中文分词，原始数据集由台湾中央研究院、香港城市大学、北京大学及微软亚洲研究院共同提供，其中前二者是繁体中文，后二者是简体中文，本实践使用北京大学提供的简体中文数据集，共20201条数据。由于原始数据集中缺少验证集，因此我们从训练集中划分1000条数据作为验证集，最终训练集18056条数据，验证集1000条数据，测试集1945条数据。
